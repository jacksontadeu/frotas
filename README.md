
# Sistema de Controle de Frotas

- Sistema para controlar as manutenções preventivas de uma frota de veículos.

- Cadastro de bases
- Cadastro de veículos
- Cadastro de manutenções
- Emissão do alerta para manutenções


# Tecnologias utilizadas

- Java 21 LTS
- Spring Boot
- Spring Web
- Spring Data
- Spring OpenFeign
- Spring OpenApi
- Thymeleaf
- Banco de Dados MySQL
